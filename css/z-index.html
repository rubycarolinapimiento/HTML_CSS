para quee sirve

Cuando se colocan elementos, pueden superponerse a otros elementos. La z-indexpropiedad especifica el orden de pila de un elemento (qué elemento debe colocarse delante o detrás de los demás). Un elemento puede tener un orden de pila positivo o negativo z-index solo funciona en elementos posicionados (posición: absoluta, posición: relativa, posición: fija o posición: pegajosa) y elementos flexibles (elementos que son hijos directos de visualización: elementos flexibles).
